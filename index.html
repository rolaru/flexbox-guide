A practical guide to CSS Flexbox

A simple example

1
2
3
<div class="simple-container">
    <div class="flex-item">1</div>
    <div class="flex-item">2</div>
    <div class="flex-item">3</div>
</div>
                        
.flex-item {
    background-color: #3a5484;
    line-height: 15rem;
    text-align: center;
    color: #fff;
    font-size: 5rem;
    box-sizing: border-box;
    border: 5px solid #000;
}

.simple-container {
    display: flex;
}

.simple-container .flex-item {
    width: 33%;
    height: 15rem;
}

The flex-shrink and flex-basis properties

1
2
3
<div class="flex-shrink-container">
    <div class="flex-item">1</div>
    <div class="flex-item">2</div>
    <div class="flex-item">3</div>
</div>
                        
.flex-item {/* Same as before... */}

.flex-shrink-container {
    display: flex;
    align-items: stretch;
}

.flex-shrink-container .flex-item {

    /* Represents the default
       size of the item before
       space is distributed. */
    flex-shrink: 1;

    flex-basis: 50%;
}

.flex-shrink-container .flex-item:nth-child(2) {
    flex-shrink: 2;
}
